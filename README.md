## SOBRE EL PROYECTO
Trabaje en un tamaño de pantalla de 1080x1920 con Visual Studio Code.
Intenete diseñar una página lo más completa y funcional posible, con una estetica que me gusta mucho y que me representa. Trabaje con distintos htmls: Index, Sobre Mi, Proyectos y Contacto.
Use pequeñas animaciones para los heros de cada página que le dan un distintivo a cada una.
El Index esta adaptado para mobile.
Use un poquito que Javascript con ayuda de la IA para cosas como el formulario y el menú hamburguesa en mobile.
